# Lynx Framework

Lynx is a high-performance HTTP web framework written in S. Lynx is designed for building REST APIs, web applications, and microservices.

## Directory Structure

- `src/`: Contains the source code for the Lynx framework.
- `bin/`: Stores compiled binaries.
- `pkg/`: Contains reusable packages and libraries.
- `config/`: Configuration files for the framework.
- `logs/`: Log files generated during runtime.
- `tests/`: Unit and integration tests.

## Getting Started

1. Clone the repository.
2. Navigate to the `src/` directory to start development.
3. Use the `tests/` directory for writing and running tests.
